## 🚀 Installation

```bash
# Cloner le projet
git clone https://github.com/matrixunplgd/agro-senegal.git
cd agro-senegal

# Installer les dépendances
pip install -r requirements.txt

# Lancer l'application
streamlit run app.py
```
